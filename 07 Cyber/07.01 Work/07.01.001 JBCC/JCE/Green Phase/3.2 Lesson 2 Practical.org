#+title: CVE ID: CVE-2022-34691

Windows Version:
Windows 7, 8, 10, 11
Windows Server 2008, 2012, 2016, 2019, 2022

CVE Overview: A Zero Day vulnerability. Active Directory Domain Services Elevation Vulnerability

Impact of the CVE: Allows for privilege elevations through the use of a certificate that is created before the account.

Fix for the CVE: Patch the systems. Audit events allows for mitigation of these particular events.
